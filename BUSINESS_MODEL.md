# Business Model — Goalnity

## 1. Overview

Goalnity는 **축구 구장 대관 · 팀 매칭 · 커뮤니티**를 통합한 플랫폼으로,
구장 운영자와 축구 플레이어를 연결하는 **양면 시장(Two-sided Market)** 구조를 기반으로 한다.

---

## 2. Target Market

### 고객 세그먼트

| 세그먼트 | 설명 |
|---------|------|
| **플레이어 (B2C)** | 축구를 즐기는 일반 사용자 (아마추어, 직장인 리그, 동호회) |
| **구장 운영자 (B2B)** | 풋살장 · 축구장을 운영하는 개인 또는 사업체 |
| **팀/동호회 (B2B2C)** | 정기적으로 경기를 운영하는 아마추어 팀 |

### 시장 규모 (TAM/SAM/SOM)

| 구분 | 정의 |
|------|------|
| **TAM** | 국내 생활체육 스포츠 시설 대관 시장 전체 |
| **SAM** | 국내 풋살/축구 구장 대관 + 매칭 플랫폼 시장 |
| **SOM** | MVP 론칭 후 수도권 기반 초기 점유 목표 |

---

## 3. Value Proposition

### 플레이어
- 구장 실시간 검색 · 예약 (지역/날짜/가격 필터)
- 실력 · 지역 기반 팀 매칭으로 경기 상대 빠르게 찾기
- 커뮤니티에서 용병 모집, 팀 구성 간소화

### 구장 운영자
- 예약 관리 자동화 (빈 슬롯 노출, 취소 처리)
- 플랫폼 통해 신규 고객 유입
- 리뷰·평점 기반 신뢰도 구축

---

## 4. Revenue Streams

### 수익 모델

| # | 모델 | 설명 | 적용 시점 |
|---|------|------|---------|
| **R-01** | 대관 중개 수수료 | 구장 예약 완료 시 결제금액의 N% 수수 | MVP |
| **R-02** | 구장 운영자 구독 (Pro) | 월정액 구독 시 노출 우선순위 상향, 통계 대시보드 제공 | V2 |
| **R-03** | 매칭 프리미엄 | 고급 매칭 필터 (실력 등급, 팀 평점 기준) 유료 개방 | V2 |
| **R-04** | 광고 (구장/장비 브랜드) | 플랫폼 내 타겟 배너 광고 | V3 |
| **R-05** | 커뮤니티 프리미엄 | 팀 프로필 꾸미기, 공식 리그 참가권 | V3 |

### 수수료 구조 (R-01 기준)

```
예약 금액 × 수수료율 = Goalnity 수익

- MVP 목표 수수료율: 5~8%
- 구장 운영자 정산: 예약 완료 후 D+3 영업일
- 취소 정책: 경기 24시간 전 취소 시 전액 환불, 이후 부분 환불
```

## 5. Cost Structure

| 비용 항목 | 설명 |
|---------|------|
| **인프라** | 서버 (NestJS API), DB, CDN, 파일 스토리지 |
| **결제 시스템** | PG사 연동 수수료 (토스페이먼츠 등) |
| **마케팅** | 초기 구장 운영자 유치, SNS 광고 |
| **운영** | CS, 데이터 검증, 구장 정보 관리 |
| **개발** | 기능 개발 및 유지보수 |

## 6. Key Metrics (목표 지표)

| 단계 | 지표 | 목표 |
|------|------|------|
| MVP | 등록 구장 수 | 수도권 50개+ |
| MVP | 월간 예약 건수 | 300건+ |
| V2 | GMV (총 거래금액) | 월 5,000만원+ |
| V2 | 매칭 성사율 | 70%+ |
| V3 | 구장 운영자 Pro 전환율 | 30%+ |

## 7. Go-to-Market Strategy

### Phase 1 — 공급 확보 (구장 운영자 유치)
- 수도권 풋살장 직접 영업 및 무료 입점 제공
- 예약 관리 툴 무료 사용으로 진입 장벽 낮춤

### Phase 2 — 수요 확보 (플레이어 유입)
- 직장인 리그, 동호회 커뮤니티 타겟 SNS 마케팅
- 첫 예약 할인 쿠폰 제공

### Phase 3 — 네트워크 효과
- 매칭 기능으로 플레이어 간 연결 강화
- 리그/토너먼트 기능으로 재방문율 향상

## 8. Competitive Landscape

| 경쟁사 | 강점 | 약점 | Goalnity 차별점 |
|--------|------|------|----------------|
| 네이버 예약 | 트래픽 | 축구 특화 기능 없음 | 매칭 + 커뮤니티 통합 |
| 카카오 스포츠 | 브랜드 인지도 | UX 복잡 | 심플한 UX, 축구 집중 |
| 개별 구장 앱 | 로열티 | 다수 앱 관리 불편 | 단일 플랫폼 |

---

## 9. Roadmap

| 시점 | 목표 |
|------|------|
| **MVP (0~3개월)** | 구장 대관 예약 + 결제, 수도권 50개 구장 |
| **V2 (3~6개월)** | 팀 매칭, 구장 운영자 Pro 구독, 리뷰 시스템 |
| **V3 (6~12개월)** | 커뮤니티, 아마추어 리그/토너먼트, 광고 플랫폼 |

---

## 10. Risks

| 위험 | 대응 |
|------|------|
| 구장 운영자 유치 어려움 | 초기 무료 제공 + 직접 영업 |
| 결제 분쟁 / 취소 처리 | 명확한 취소 정책, CS 대응 매뉴얼 |
| 경쟁사 진입 | 축구 특화 매칭·커뮤니티로 락인 효과 |
| 계절성 수요 변동 | 실내 풋살장 중심으로 연중 수요 확보 |
