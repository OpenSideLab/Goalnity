# ⚽ Goalnity

> 축구를 더 쉽게, 함께

## 소개

**Goalnity**는 축구 구장 대관부터 팀 매칭, 커뮤니티까지 한 곳에서 관리할 수 있는 축구 통합 플랫폼입니다.
구장을 찾고, 팀을 꾸리고, 경기를 즐기는 모든 과정을 Goalnity에서 해결하세요.

---

## 주요 기능

### 🏟️ 구장 대관
- 지역별 축구장 검색 및 실시간 예약
- 예약 현황 캘린더 확인
- 예약 알림 및 일정 관리
- 구장 리뷰 및 평점

### ⚽ 팀 매칭
- 실력 / 지역 기반 팀 매칭 시스템
- 매칭 요청 · 수락 · 거절
- 매칭 결과 기록 및 전적 통계
- 레이팅 기반 실력 등급 시스템

### 👥 커뮤니티
- 팀원 / 용병 모집 게시판
- 경기 후기 및 리뷰 공유
- 팀 랭킹 및 개인 레이팅
- 공지사항 및 이벤트 관리

## 시작하기

> 기술 스택 및 환경 설정 방법은 추후 업데이트될 예정입니다.


## 기여하기

Goalnity는 오픈소스 프로젝트입니다. 기여를 환영합니다!

1. 이 저장소를 Fork 합니다
2. 새 브랜치를 생성합니다 (`git checkout -b feature/새기능`)
3. 변경사항을 커밋합니다 (`git commit -m 'feat: 새기능 추가'`)
4. 브랜치에 Push 합니다 (`git push origin feature/새기능`)
5. Pull Request를 생성합니다

버그 리포트나 기능 제안은 [Issues](https://github.com/OpenSideLab/Goalnity/issues)를 이용해 주세요.
